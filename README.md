## Quick Start Guide

```bash
# Clone the repository (it's going to be yuge!)
git clone https://github.com/TrumpPersonaAI/Trump-Persona-AI.git

# Install dependencies (we have the best dependencies)
npm install

# Start the development server (it's going to be fantastic)
npm run dev
```

```bash

### Project Structure
```
trump-persona/
├── src/
│   ├── components/        # Tremendous UI components
│   │   ├── ChatMessage/  # Message components
│   │   ├── ChatInput/    # Input components
│   │   └── Loading/      # Loading animations
│   ├── lib/              # Utility functions
│   │   ├── constants.ts  # Trump quotes & stickers
